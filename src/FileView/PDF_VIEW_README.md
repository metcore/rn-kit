# PdfView Component

`PdfView` adalah sebuah komponen React Native sederhana untuk menampilkan file PDF menggunakan library [`react-native-pdf`](https://github.com/wonday/react-native-pdf). Komponen ini membungkus `Pdf` dengan container fleksibel sehingga mudah digunakan di berbagai layar.

---

## Instalasi

Pastikan `react-native-pdf` sudah terpasang di proyek:

```bash
npm install react-native-pdf
# atau
yarn add react-native-pdf
```
