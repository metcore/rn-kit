## Props
| Prop    | Tipe   | Default | Deskripsi                |
| ------- | ------ | ------- | ------------------------ |
| `name`  | string | null    | Jenis Icon |
| `size`  | number | 24      | Ukuran icon dalam piksel |
| `color` | string | "#000"  | Warna icon               |
| `style` | object | `{}`    | Gaya tambahan (optional) |
