# Badge

## Usage

```js
import { Badge } from '@herca/ui-kit';

export default function BadgeScreen(){
    return(
      <Badge
        value="Complete"
        color="Primary"
        size="warning"
      />
    )
}
```

## Props

| props name           | mandatory | types   | default                                                  |
| -------------------- | --------- | ------- | -------------------------------------------------------- |
| value             |          | string | null                                            |
| color              |         | string    | null                                              |
| size       |           | string  | medium       |
